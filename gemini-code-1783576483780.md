# ☕ Coffee POS & Bakery 

ระบบ Point of Sale สำหรับร้านกาแฟและเบเกอรี่ พร้อมระบบสะสมแต้ม

## 🌟 ฟีเจอร์หลัก
- 🌓 สลับโหมดหน้าจอ (Light Mode / Dark Mode)
- 🛒 ระบบขายกาแฟและเบเกอรี่
- 🎁 ระบบสะสมแต้ม (ทุก 50 บาท = 1 แต้ม)
- 🔐 ระบบ Login/Register โดยใช้ Supabase
- ☁️ พร้อม Deploy ขึ้น Vercel ทันที

## ⚙️ วิธีการตั้งค่าฐานข้อมูล (Supabase)

1. สมัครใช้งานและสร้างโปรเจกต์ใหม่ที่ [Supabase](https://supabase.com/)
2. ไปที่เมนู **Authentication** -> Providers ยืนยันว่าเปิด Email auth ไว้ (แนะนำให้ปิด "Confirm email" ชั่วคราวเพื่อให้เทสง่ายขึ้น)
3. ไปที่เมนู **SQL Editor** และรันคำสั่ง SQL ด้านล่างเพื่อสร้างตารางเก็บแต้มสะสม:

```sql
create table public.profiles (
  id uuid references auth.users not null primary key,
  points integer default 0
);

-- ปรับ Policy เพื่อให้ User แก้ไขแต้มตัวเองได้ (เพื่อความรวดเร็วในการทดสอบ)
alter table public.profiles enable row level security;
create policy "Users can update own profile." on profiles for update using (auth.uid() = id);
create policy "Users can select own profile." on profiles for select using (auth.uid() = id);
create policy "Users can insert own profile." on profiles for insert with check (auth.uid() = id);
